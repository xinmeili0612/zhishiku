"<div style=\"margin-bottom: 2.5rem;\">\r\n  <h3 style=\"font-size: 1.5rem; font-weight: 700; color: #1f2937; margin-bottom: 1rem; margin-top: 0; padding-bottom: 0.5rem; border-bottom: 2px solid #e5e7eb;\">1 小说转短剧的基本规范</h3>\r\n  \r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(1) 删除掉小说创作中，对人物形象、心理活动的刻画描写。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">将上述内容，转化为人物的台词、动作或表情，通过以上方式甚至穿着打扮这些直观可视化信息来达到原本目的。</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(2) 删除掉小说创作中，对环境氛围的烘托描写。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">只需交待清楚时间、地点，并用可视化的语言，概括性交待必要的环境和氛围信息。</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(3) 删除掉小说创作中，对事件推进的连接。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">通俗来讲，就是对于事件描述，无需将每一步都讲出来，而是抽出与主线剧情或核心情绪有关的事件。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">举例来说，主角去酒店参加同学聚会，那么在酒店门口被服务员拦下嘲讽，主角反打脸，这个小事件要不要插入，就要结合剧本本身的主线剧情和核心情绪来考虑。如果加入这个小事件，是为了引出主线剧情的人物、或者对服务于核心情绪点的人物或事件有铺垫渲染的作用，那就可以考虑加入，反之则不建议添加。</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(4) 精炼人物对话，台词贴合人设。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">剧本中的台词一直是重中之重，是写好一个剧本的最基础要求。首先要讲的，就是台词一定要贴合人设！也就是说，什么样的人，该说什么样的话？以什么语气说？该做什么事，以什么理由去做！这些可能看剧的人感觉不到，但创作者必须清楚且合理，否则呈现出的成品，就给人一种人设撕裂的感觉！</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">接下来再说台词的意义，对于时常一共一两分钟的短剧剧本而言，每一句台词都必须经过反复推敲，做到用最简单的语言，表述出尽可能清晰、激烈的信息点。台词的目的是塑造人物形象、推动剧情进程、加强情绪烈度、交待必要信息、递进人物关系。如果这句台词没有包含关键信息点，那就要考虑它是否有必要存在了。</p>\r\n  </div>\r\n</div>\r\n\r\n<div style=\"margin-bottom: 2.5rem;\">\r\n  <h3 style=\"font-size: 1.5rem; font-weight: 700; color: #1f2937; margin-bottom: 1rem; margin-top: 2rem; padding-bottom: 0.5rem; border-bottom: 2px solid #e5e7eb;\">2 人物小传</h3>\r\n  \r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(1) 存在意义。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">人物小传的存在，是制作组在选择演员时的一个重要参考指标，所以要尽可能的说清必要信息，人物形象以及所要表现出的荧幕感觉。另外也是演员快速了解这个人物性格的重要渠道，所以对于主要角色，甚至要讲清楚其前因后果，性格的转化节点等必要信息。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">比如女主，本来是什么样的性格，生长在怎样的生活环境中造成了她什么样的性格，现在拥有什么样的生活以及隐藏身份，之后会因为什么事而变得怎样？</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(2) 包含元素。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">首先是这个人的性别、年龄、形象气质，如有脸谱化的形象特征或者标签化的人物特征，也要写出来。其次是主角、反派、还是配角？与主要角色是什么样的人际关系，这些都要讲清楚。如果是与主要角色有情感线的，要简要说清楚情感线，在主线中有哪几个关键点，人物性格有变化的要标注清楚在哪个事件引起性格变化，变化后变成什么样。</p>\r\n  </div>\r\n</div>\r\n\r\n<div style=\"margin-bottom: 2.5rem;\">\r\n  <h3 style=\"font-size: 1.5rem; font-weight: 700; color: #1f2937; margin-bottom: 1rem; margin-top: 2rem; padding-bottom: 0.5rem; border-bottom: 2px solid #e5e7eb;\">3 情节要素</h3>\r\n  \r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">（1）故事背景。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">这个在短剧中，我们通常是交待主角的背景，比如战神文的逼格渲染，比如虐恋文或甜宠文的一夜情。</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">（2）起因。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">也就是我们说的事件切入点，比如市场上的绿帽元素、妻女被辱的强情绪元素、开局给丈母娘倒洗脚水等。</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">（3）铺垫。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">这里的铺垫指的是，告诉观众，主角随时有对反派进行降维打击的能力，也就是对观众情绪期待值的铺垫！</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">（4）钩子。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">在短剧本中，免费集数尽量做到每一集末尾，都要有一个吸引观众继续看下去的钩子，即便付费之后，最多两集也要留有钩子。基本方法就是让观众觉得即将发生反转、或事件再次升级。</p>\r\n  </div>\r\n</div>\r\n\r\n<div style=\"margin-bottom: 2.5rem;\">\r\n  <h3 style=\"font-size: 1.5rem; font-weight: 700; color: #1f2937; margin-bottom: 1rem; margin-top: 2rem; padding-bottom: 0.5rem; border-bottom: 2px solid #e5e7eb;\">4 基础模版</h3>\r\n  \r\n  <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem;\">一个完整的短剧剧本，应该包含以下五点内容：</p>\r\n  \r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(1) 故事梗概。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">简要的说一下，这个剧本是要将一个什么故事。三百字以内就行。</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(2) 故事亮点。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">剧本的题材类型，融合了哪些元素。比如：团宠，闪婚、复仇等。</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(3) 人物小传。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">参考上面人物小传的说明内容。</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(4) 分集梗概。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">一两句话讲清楚，每一集要讲的主要内容。</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(5) 剧本正文。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">参考下方剧本正文的基本格式要求。</p>\r\n  </div>\r\n</div>\r\n\r\n<div style=\"margin-bottom: 2.5rem;\">\r\n  <h3 style=\"font-size: 1.5rem; font-weight: 700; color: #1f2937; margin-bottom: 1rem; margin-top: 2rem; padding-bottom: 0.5rem; border-bottom: 2px solid #e5e7eb;\">5 剧本正文基本格式</h3>\r\n  \r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(1) 场次。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">每一集的场次、人物、时间、地点写清楚。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">例如：</p>\r\n    <div style=\"background: #f9fafb; padding: 1rem; border-radius: 0.5rem; margin-left: 1rem; margin-bottom: 1rem; border-left: 3px solid #3b82f6;\">\r\n      <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 0.95rem; font-family: monospace;\">第一集</p>\r\n      <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 0.95rem; font-family: monospace;\">1-1 日内 酒店-大堂</p>\r\n      <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 0.95rem; font-family: monospace;\">人物：张三 李四</p>\r\n      <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0; font-size: 0.95rem; font-family: monospace;\">1-2 夜外 酒店-前台</p>\r\n      <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0; font-size: 0.95rem; font-family: monospace;\">人物：王五 赵六 保安2</p>\r\n    </div>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(2) 人物。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">这一场要出场的所有人物，人物信息要清晰统一，有名字的一律写清名字，方便统筹归纳；</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">主要人物以外的环境人物（比如保安、拍卖会来客、宴会宾客等），需要几个、什么身份都要标注清楚，其中如果有动作或台词的，要用甲、乙、丙等区分；</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">人物有特殊需求的，要标注清楚，方便道具准备。比如湿漉漉的头发、戴着什么发夹等。</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(3) 时间。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">标注清楚日、夜、黄昏等，如需要雨、雪、大风等，也要标注清楚。</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(4) 场景。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">室内还是室外，同时详细交代拍摄场景，如酒店包间、野外树林、林家别墅等。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem; font-weight: 600; color: #dc2626;\">【与人物一样，同一个地点一定要做到名称的前后统一。比如小明和大明是兄弟俩，住在一起，那么前面写的是小明家，全本都要写小明家，不能后面变成了大明家！这样统筹会产生混乱】</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">场景描述要尽可能清楚，比如发生在卧室，那就是\"小明家-小明卧室\"、\"小明家-大明卧室\"、\"医院-走廊\"、\"医院-病房\"等等。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">一旦发生场景转换，那么就要独立为一场。比如在餐厅门口和服务员对话，这是一场，那么进了餐厅，就是另外一场了！</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(5) 正文。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">台词、动作、场景要单独成一行，不同人物的台词要单独成行。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">动作前需要添加△符号。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">每集集数在500-700字间。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">人物说台词时的情绪、表情等，在人物冒号后加括号进行备注。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">如：沈初晴(浅笑)</p>\r\n  </div>\r\n\r\n  <div style=\"margin-bottom: 1.5rem;\">\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; font-weight: 600;\">(6) 特殊场景。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\"><strong>os：</strong>角色的内心戏。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">例：沈初晴OS：</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\"><strong>Vo：</strong>画外音。人未到，声音先到。或者是打电话的时候，只能听见声音，看不见人的时候。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">例：</p>\r\n    <div style=\"background: #f9fafb; padding: 1rem; border-radius: 0.5rem; margin-left: 1rem; margin-bottom: 1rem; border-left: 3px solid #3b82f6;\">\r\n      <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 0.95rem; font-family: monospace;\">△ 跟班摁着沈初晴，李雪撸起袖子准备扇向沈初晴</p>\r\n      <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 0.95rem; font-family: monospace;\">萧南vo：谁敢动沈氏集团的大小姐！</p>\r\n      <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0; font-size: 0.95rem; font-family: monospace;\">△ 萧南走进来</p>\r\n    </div>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">梦境、幻象、闪回等，标清楚【闪回开始】，结束时标注【闪回结束】。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">多场景连续闪回或互切，要用\"/\"分隔。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0.5rem; font-size: 1rem; padding-left: 1rem;\">内心独白与实际台词并存时，各占一行。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 1rem; font-size: 1rem; padding-left: 1rem;\">场景互切，比如打电话中的双方，可以放在同一场，标清楚场景互切就行。</p>\r\n    <p style=\"line-height: 1.8; color: #374151; margin-bottom: 0; font-size: 1rem; padding-left: 1rem; font-weight: 600; color: #dc2626;\">正文8-10集为第一个付费卡点，后续剧情，每二十集左右设计一个付费卡点，在付费卡点集的末尾，需标红备注【付费卡点】</p>\r\n  </div>\r\n</div>\r\n\r\n"
